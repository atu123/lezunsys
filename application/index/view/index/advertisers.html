    关于广告商广告页面的规定

    1. 放置广告信息的弹窗/点击页面必须是真实页面，不得以任何形式将代码放置于非法页面上，也就是不能通过javascript、ifram、js等已知或未知的方式在其它页面调出，如有此类情况一经查实，我们将撤销该广告并清除该广告剩余投放金额。 　　
    2. 放置广告信息的弹窗页面不能再有弹出窗口，否则广告商广告将被暂停并清除该广告剩余投放金额。 　　
    3. 不允许弹出超过一次以上将广告页面设为主页和设为收藏的对话框，否则广告商广告将被暂停并清除该广告剩余投放金额。 　　
    注： 以上为乐樽联盟判定非法广告页面的标准并由发布之日起生效，并严格执行。如发现以上任何违规行为，将撤销其广告项目、没收广告投放金额，情节严重者删除 ID 并永不通过审核。 　　
    乐樽有权对以上判定广告商非法广告页面的标准根据实际情况作出相应的修改、补充和完善。 　　
    乐樽保留最终解释权。
